# Clean Architecture

## Workflow

1. Identify domain, application, and infrastructure layers.
2. Ensure dependencies point inward.
3. Keep interfaces in inner layers.
4. Isolate framework and IO concerns.
5. Flag violations with concrete fixes.

## Output Expectations

Provide:
- Boundary checks
- Violations found
- Suggested fixes
