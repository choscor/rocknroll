# Log Analysis

## Workflow

1. Identify the relevant log source and timeframe.
2. Extract errors, warnings, and patterns.
3. Correlate with recent changes or actions.
4. Propose likely causes and next steps.

## Output Expectations

Provide:
- Key log excerpts (summarized)
- Likely causes
- Recommended actions
