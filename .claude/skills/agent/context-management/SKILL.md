# Context Management

## Workflow

1. Start with high-signal files (README, AGENTS, key modules).
2. Use `rg` to locate relevant code instead of broad reads.
3. Load only the necessary sections.
4. Summarize findings and track assumptions.
5. Avoid duplicating large chunks of text.

## Notes

- Prefer references over full file dumps.
- Keep summaries actionable.

## Output Expectations

Provide:
- Files/sections read
- Key findings
- Assumptions or open questions
