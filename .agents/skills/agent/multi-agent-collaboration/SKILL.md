# Multi-Agent Collaboration

## Workflow

1. Define roles (planner, coder, reviewer, tester).
2. Assign clear scopes to each role.
3. Avoid overlapping edits without analysis.
4. Reconcile findings before final changes.

## Output Expectations

Provide:
- Role assignments
- Scope boundaries
- Handoff notes
