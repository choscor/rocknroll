# Safe Code Modification

## Workflow

1. Identify the smallest change that solves the issue.
2. Avoid unrelated refactors.
3. Preserve interfaces and contracts.
4. Add tests when behavior changes.
5. Summarize risks and validation.

## Output Expectations

Report:
- Changes made
- Risk assessment
- Validation results
