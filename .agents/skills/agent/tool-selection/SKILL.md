# Tool Selection

## Workflow

1. Choose the minimal tool to get the needed signal.
2. Prefer `rg` for search and existing scripts for checks.
3. Avoid expensive or destructive commands.
4. Escalate permissions only when required.

## Output Expectations

Provide:
- Selected tools/commands
- Rationale for choices
