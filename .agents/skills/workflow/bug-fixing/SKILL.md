# Bug Fixing

## Workflow

1. Restate the issue and expected behavior.
2. Reproduce with a test or manual steps.
3. Identify the smallest fix.
4. Implement and re-run the reproduction.
5. Add a test if the bug was untested.

## Output Expectations

Report:
- Repro steps
- Fix summary
- Verification results
