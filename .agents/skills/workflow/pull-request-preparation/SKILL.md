# Pull Request Preparation

## Workflow

1. Confirm scope and review all diffs.
2. Ensure tests/lint/build checks are run as applicable.
3. Summarize changes, risks, and verification.
4. Note any follow-ups or known gaps.

## Output Expectations

Provide:
- Summary of changes
- Tests/commands run
- Risk notes and follow-ups
