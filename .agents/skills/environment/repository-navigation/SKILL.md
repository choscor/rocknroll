# Repository Navigation

## Workflow

1. Start with `README.md`, `AGENTS.md`, and key config files.
2. Use `rg` to find symbols and usage.
3. Open only relevant files/sections.
4. Summarize findings and next steps.

## Output Expectations

Provide:
- Files searched/read
- Key locations found
- Assumptions or open questions
